  • Rainbow Six API
    • https://github.com/vince1444/r6api
    • A lightweight API built in vanilla Javascript with 700+ downloads
    • Used to fetch a ton of different statistics pertaining to a player
    • Developed with a focus on usability and ease-of-use with documentation

My idea for this project came to me when I was playing the game Rainbow Six Siege. I wanted something that could pull player statistics so that me and my friends could see other players' ranks and such. This was my first big project after learning the basics of Javascript. All I knew how to do was a very simple GET/POST request and I dove into the deep end trying to figure this project out. It took me weeks of googling to really get some results, but in the end it made me a way better developer than I previously was. I attribute this project to my love of Javascript and success in hackathons/other projects.
